Quizdrill

OpenFacts, the Open Source Knowledge Database

Quizdrill is a learning-by-testing program to learn quickly, mostly memorizing tasks like vocabulary. Quizdrill supports multiple choice, simple quiz as well as flashcard testing. Although still quite immature, Quizdrill asks questions which have been answered right more often then others, less often to improve learning efficiency. Quizzes can be easily created by edition simple text files or automatically built from Infobox-style templates of Wikipedia dumps (or other MediaWikis and even Semantic MediaWikis).

Quizdrill should run on Linux, Mac/OSX, Unixes and Window. To run quizdrill you need Python (tested with 2.4), PyGTK (tested with 2.8.6) and EasyInstall. For bug reports and freature requests see the Quizdrill project page. Quizdrill is released under the GNU GPL either version 2 of the License, or (at your option) any later version.
