Quizdrill

OpenFacts, the Open Source Knowledge Database

Quizdrill is a learning-by-testing program to learn quickly, mostly memorizing tasks like vocabulary. Quizdrill supports multiple choice, simple quiz as well as flashcard testing.

Quizdrill should run on Linux, Mac/OSX, Unixes and Window. To run quizdrill you need python (tested with 2.4) and pygtk (tested with 2.8.6). For bug reports and freature requests see the Quizdrill project page. Quizdrill is released under the GNU GPL either version 2 of the License, or (at your option) any later version.
